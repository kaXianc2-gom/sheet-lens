# SheetLens

表格数据筛选工具 — 单文件 HTML，双击即用。

拖拽上传 Excel 表格，自动识别列名，多维度筛选，地图可视化，一键导出。

## 功能

- 📂 **拖拽上传** — 支持 .xlsx 格式，自动识别列名，可多文件合并
- 🔍 **多维度筛选** — 支持按任意列关键词组合筛选
- 🗺 **地理分布** — 中国地图热力图，点击省份快速定位
- 📈 **趋势图表** — 数据分布饼图
- 💾 **CSV 导出** — 筛选结果一键导出
- 🚫 **零安装** — 纯 HTML+JS，浏览器打开即用

## 使用方式

1. 下载 `index.html`
2. 双击在浏览器中打开
3. 拖拽你的 Excel 文件到上传区
4. 开始筛选分析

## 兼容格式

工具自动识别列名（关键词匹配），不依赖列序。支持：

- 国考标准格式（29列）
- 省考标准格式
- 带合并单元格说明行的变体格式（自动跳过）

## 开发

```bash
# 从 Excel 文件预构建数据（可选）
python build_data.py

# 依赖
pip install openpyxl
```
